YouTube often blocks embeds when the page is opened from disk (file://). To get Beograd and other YouTube videos to play in the overlay, run the site from a local server:

  In Terminal, go to this folder and run:

    python3 -m http.server 8000

  Then open in your browser:   http://localhost:8000/index.html

  Or use the "Live Server" extension in VS Code / Cursor to open the folder and serve the page.
